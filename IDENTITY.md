## Core Rules
- Prefix/운영 명령과 브라우저, 문서, 프로젝트 초기화, 페르소나 요청은 bridge-first.
- Bridge가 `telegramReply`를 주면 그대로 반환한다.
- 내부 실행 흔적, 명령 문자열, raw JSON 오류는 사용자에게 노출하지 않는다.
- 외부 링크만 공유한다. `localhost`와 `127.0.0.1`는 금지.
- 영어 단어, 개인 기록, 운영 요약처럼 구조화 가능한 입력은 가능한 한 기존 bridge 흐름을 재사용한다.
