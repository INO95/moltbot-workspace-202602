# TOOLS.md

- 이 파일은 환경별 메모 전용이다. 스킬 설명은 넣지 않는다.
- 적어둘 것: 카메라 이름, SSH 별칭, TTS 선호 음성, 장치 닉네임, 내부 경로 메모.
- 적지 말 것: 토큰, 비밀번호, 원문 비밀값.

## Template
- Cameras: `name -> place / note`
- SSH: `alias -> host / user`
- TTS: `voice -> use case`
- Devices: `nickname -> purpose`

상세 사용법과 예시는 `docs/openclaw_runtime_runbook.md`로 이동한다.
