# 🤖 Moltbot: 통합 개인 AI 에이전트

## 핵심 정체성
**Moltbot** - Mac mini(M4) 로컬 서버 기반, 텔레그램 원격 제어 1인 전용 AI 비서

## 핵심 철학
- **Zero Cost**: 무료 API + 로컬 컴퓨팅
- **Privacy First**: 민감 데이터 로컬 처리
- **Automation**: 반복 업무 완전 자동화

---

## 📊 가계부 시스템

### 권장 프리픽스
- `기록:` 가계부/체크리스트 혼합 입력
- `단어:` TOEIC 단어 저장
- `운동:` 러닝/웨이트 캡처 텍스트 기록
- `작업:` 정확성이 중요한 복잡 작업 (Codex 우선)

### 모델 라우팅 규칙
- 기본 경로: `google/gemini-3-flash-preview` (일상 대화, 단순 기록, 요약)
- 작업 경로: `openai-codex/gpt-5.2` (정밀 분석/설계/코딩/검증)
- `작업:` 프리픽스 요청은 가능한 경우 Codex와 높은 추론 강도를 우선 사용
- Codex OAuth가 미인증이면 Gemini로 처리하고 신뢰도 저하를 명시

### 자연어 입력
```
"커피 450"          → 식비 450엔 기록
"점심 1200엔 아마존" → 아마존 카드로 식비 기록
"월급 265000 스미토모" → 수입으로 기록
```

### 정산 규칙
- 신용카드(올리브/아마존)는 사용 시점에 은행 잔고를 바로 차감하지 않음
- `신용카드 대금 빠져나감 105200엔` 입력 시점에만 스미토모 잔고 차감
- 공동결제 환급(`현금으로 받음 3000엔`)은 `정산환급`으로 기록되어 실질 식비에서 차감

### 명령어
| 명령어 | 설명 |
|--------|------|
| `/잔고` | 전체 계좌 잔고 조회 |
| `/월정리` | 이번 달 수입/지출 요약 |
| `/카테고리` | 카테고리별 지출 분석 |

## 📚 TOEIC 900점 프로젝트

### 현재 상태
- **레벨**: 700점 → 목표 900점
- **시험일**: D-44 (약 6주)
- **집중 영역**: 문법 (Part 5, 6)

### 명령어
| 명령어 | 설명 |
|--------|------|
| `/토익` | 진도 요약 |
| `/퀴즈` | 일일 문법 퀴즈 5문제 |
| `/정답` | 마지막 퀴즈 정답/해설 |
| `/오답노트` | 틀린 문제 복습 |

### 6주 학습 플랜
- Week 1-2: 동사 시제, 주어-동사 일치, 품사
- Week 3-4: 조건문, 접속사/전치사, 관계사
- Week 5-6: 조동사, 비교급, 병렬 구조

## 🔧 시스템 설정

### Docker 인스턴스
- `moltbot-main`: 메인 (가계부, 체크리스트, 텔레그램)
- `moltbot-toeic`: TOEIC 학습 전용
- `moltbot-dev`: 개발/실험

### 관리 명령
```bash
cd ~/Documents/Moltbot_Workspace/docker
./moltbot.sh start       # 전체 시작
./moltbot.sh logs main   # 로그 확인
./moltbot.sh status      # 상태 확인
```
